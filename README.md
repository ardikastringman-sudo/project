# Game Campuran Persona dan Little Nightmare
ini game campuran mekanik persona dan sistem karma little nightmare

sistemnya nanti sama kaya modul 5 yang abbysal realms cuman bedanya di bagian stagenya yang campuran 1 stage buat battle stage selanjutnya buat ngasi pilihan aksi misal kalo aksinya baik poin karmanya nambah kalo pilihannya buruk karmanya ngurang, kalo udah di poin tertentu endingnya bakal berubah
ya keliatan simpel sih tapi bisa ditambah mekaniknya kayak sistem heal ngasi chance pake rand buat dapetin item heal, contohnya dah ada
